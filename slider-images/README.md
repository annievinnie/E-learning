# Slider Images Folder

This folder is for storing your slider images. 

## Image Placement

For the sliders to work properly, please place your images in:
```
frontend/public/images/sliders/
```

## Image Naming Convention

Name your images as follows:

### Slider 1:
- slider1-1.jpg (or .png)
- slider1-2.jpg
- slider1-3.jpg

### Slider 2:
- slider2-1.jpg
- slider2-2.jpg
- slider2-3.jpg

### Slider 3:
- slider3-1.jpg
- slider3-2.jpg
- slider3-3.jpg

### Slider 4:
- slider4-1.jpg
- slider4-2.jpg
- slider4-3.jpg

## Notes

- Each slider can have multiple images (minimum 1, recommended 3-5)
- Supported formats: .jpg, .jpeg, .png, .webp
- Recommended image size: 800x400px or similar aspect ratio
- Images will automatically cycle through with a 4-second interval
